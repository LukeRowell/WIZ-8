# WIZ-8 #
A CHIP-8 emulator for Windows written in C++ using SFML and ImGui.

Download the latest release here: 

## Features ##
* Save states
* Pausing
* Adjustable clock speed
* Adjustable color palette
* Adjustable window size
* Native file selection dialog
